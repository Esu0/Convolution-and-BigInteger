# Convolution-and-BigInteger
任意精度整数

1億桁くらいは簡単に計算できる多倍長整数です。負の数は対応していません。

## 簡単なドキュメント
* `class ubigint<unsigned long num_base>`  
多倍長クラス。 `num_base` は進数。
   * 以下メンバ関数
   * `random()` 桁数を指定してランダムな数字を返す。
   * `dump()` とりあえず中身を表示する。

演算子がオーバーロードされているので算術演算は大体できる。  
対応済み: `+` `-` `*` `+=` `-=` `*=` `++` `--` `=` `==` `!=` `>` `<` `>=` `<=`
